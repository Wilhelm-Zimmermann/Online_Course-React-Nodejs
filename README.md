# Online_Course-React-Nodejs
## O projeto é um curso online onde pode ser adicionado módulos e aulas, apenas adminstradores podem fazer isso.

## Requisitos para rodar a aplicação
  * [Docker](https://www.docker.com/)
  * [NodeJS](https://www.nodejs.org)
  
## Dependêcias utilizadas na aplicação
* Front-End
   * Axios
   * Typescript
   * React-Router-Dom
   * React-Icons
   * Styled-Components
  
* Back-End
    * Bcrypt
    * Cors
    * Express
    * Json Web Token 
    * Pg(Postgres) como Banco de Dados
    * Tsyringe
    * Typeorm 
    * Uuid
    * Express Async Errors
    * Typescript
  
## Funcionalidades
* Módulo
  * Adicionar Módulo
  * Remover Módulo
  * Editar Módulo
  * Listar Módulos
* Aulas
  * Adicionar Aula
  * Remover Aula
  * Editar Aula
  * Listar Aulas dentro do módulo
